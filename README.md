Reverse Engineering Protection (Security Lab)

Консольная программа с реализацией базовых методов защиты программного обеспечения: антиотладка, анти-VM, контроль целостности и обфускация. Предназначена для изучения методов защиты ПО и их обхода через бинарный анализ.

Возможности:
проверка пароля пользователя
выполнение защищённого функционала при успешной аутентификации
шифрование строк (XOR)
контроль целостности кода (CRC-8)
обнаружение отладчика
обнаружение виртуальной машины
обфускация кода и управления потоком выполнения
ложные проверки и скрытая логика
Использование

При запуске программа выполняет серию проверок и затем предлагает ввод пароля.

1. Проверка среды выполнения
проверка наличия отладчика (WinAPI и системные флаги)
проверка запуска в виртуальной машине
анализ системных признаков среды

При обнаружении анализа программа завершает работу.

2. Проверка целостности
вычисление CRC-8 функции проверки пароля
сравнение с эталонным значением
при несовпадении программа завершается
3. Проверка пароля
ввод пароля пользователем
сравнение с эталонным значением
выполнение дополнительных скрытых проверок
4. Основной функционал

При успешной проверке:

выполняется полезная нагрузка (арифметические операции)
генерируется дополнительный ключ/данные
результат выводится пользователю
Принцип работы

Программа использует многоуровневую систему защиты:

распределённые проверки (антиотладка, анти-VM, CRC)
разделённая логика проверки пароля
ложные ветвления и запутанный поток выполнения
XOR-шифрование строк
inline-assembly и вставки байтов (__emit)
контроль целостности критических функций

Любое изменение бинарного кода приводит к нарушению логики проверок.

Технологии:
C / C++
WinAPI (Windows.h)
Inline Assembly (x86)
работа с процессами и памятью
работа с реестром Windows
CRC-8 контроль целостности
методы обфускации и антиотладки
